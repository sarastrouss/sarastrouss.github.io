Programming Dashboard 2.0

For my Senior Design project, I decided to rework the old Programming Dashboard by integrating it with information visualizations that focus on the number of online listeners and use of automation data at WREK while also including the necessary forms to allow Music Directors to add incoming albums to the WREK songs database. This was accomplished by using Ruby on Rails, MongoDB, Bootstrap, and d3.js. The current iteration lives on WREKtranet (intranet system for WREK Atlanta). The repo can be viewed here.

  • Role: Front End Developer
  • Client: WREK Atlanta
  • Tools: Ruby on Rails, d3.js, Bootstrap
  • Year: 2015
