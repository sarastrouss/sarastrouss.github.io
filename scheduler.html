The aim of this project was to redesign the original scheduling dashboard by limiting the number of clicks the user has to make. As there are quite a few shows at WREK, the user can filter by the type of show or search for a specific show. Currently, it is not hooked up to the WREK database. When the user clicks on a particular show, they will be able to edit that particular show. It can be viewed here.
